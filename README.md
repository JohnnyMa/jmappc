# jmappc
Cordova sample

## How to run it?
Step1:
Make your own build which outputs build.js in to /jmappc/www/js

Step2:
Setup Cordova via the following commands:
```
npm install -g cordova
cordova platform add android
cordova run android
```

More details could be found here: http://cordova.apache.org/docs/en/latest/guide/cli/index.html.
